# Internship Objective

## Overview

This project is carried out as part of the **VSD AI-Assisted PLL Design Internship**.

The objective is to understand, design, simulate, and verify the fundamental building blocks of a Phase Locked Loop (PLL) using the open-source SKY130 Process Design Kit (PDK) and open-source EDA tools.

The project emphasizes AI-assisted circuit development while ensuring that every generated circuit is verified through engineering analysis and simulation.

## Objectives

* Study PLL fundamentals and locking behavior.
* Understand the relationship between reference and feedback clocks.
* Design and simulate the Phase Frequency Detector (PFD).
* Design and simulate the Charge Pump.
* Design the Loop Filter.
* Design the Voltage Controlled Oscillator (VCO).
* Design the Frequency Divider.
* Integrate all blocks into a complete PLL.
* Perform pre-layout simulations using ngspice.
* Document all AI prompts, design decisions, debugging steps, and observations.

## Tools Used

* xschem
* ngspice
* SKY130 PDK
* Magic VLSI
* Git
* GitHub
* ChatGPT

## Expected Outcome

By the end of this internship, the repository will contain:

* Verified PLL circuit blocks
* AI-assisted design workflow
* Simulation results
* Engineering observations

* Version-controlled development history
